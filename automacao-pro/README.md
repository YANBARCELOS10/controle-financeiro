# Automação Pro — Simulador Industrial

Aplicativo didático, responsivo e instalável para treino de CLP Siemens S7-1200, entradas e saídas digitais, Ladder, IHM e diagnóstico de falhas.

## Recursos

- Esteira animada com sensor, motor e empurrador.
- Monitor de `%I`, `%Q`, `%M`, CTU e redes Ladder.
- Painel 220 Vca / 24 Vcc representado de forma didática.
- Banco de dez falhas com instrumentos virtuais, ordem de serviço, pontuação e histórico.
- Curso integrado e progresso salvo no aparelho.
- PWA offline e estrutura Capacitor para Android.

## Abrir no celular

Depois de publicar pelo GitHub Pages, abra o endereço no Chrome e escolha **Adicionar à tela inicial**.

## Gerar APK no GitHub

Abra **Actions → Gerar APK Android → Run workflow**. Ao terminar, baixe o artefato `Automacao-Pro-APK`.

## Segurança

Este aplicativo é uma simulação educacional. Ele não deve ser utilizado para comandar máquinas reais. Em campo, siga os procedimentos da empresa e as normas aplicáveis.
