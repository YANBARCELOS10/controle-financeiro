# Simulador CLP Industrial

Aplicativo didático em HTML para treinamento de automação industrial.

## Recursos

- Simulação de esteira transportadora
- Entradas e saídas digitais
- Lógica Ladder simplificada
- Partida, parada, emergência e reset
- Injeção de falhas de sensor, contatora, relé térmico e segurança
- Diagnóstico orientado para manutenção
- Registro de eventos
- Interface responsiva para celular

> Material educacional. Não substitui procedimentos de segurança, documentação da máquina ou programação validada em CLP real.
