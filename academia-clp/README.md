# Academia CLP Industrial

Aplicativo educacional em HTML para estudo de automação industrial.

## Conteúdo
- fundamentos de CLP
- ciclo de scan
- entradas, saídas e memórias
- leitura de Ladder
- selo, intertravamentos, temporizadores e contadores
- laboratório de lógica interativa
- diagnóstico de falhas de sensor, saída, motor, segurança e comunicação
- quiz com progresso salvo no navegador
- funcionamento offline por service worker

## Aviso
Material exclusivamente didático. Não deve ser usado para controlar máquinas reais nem para burlar circuitos de segurança.
